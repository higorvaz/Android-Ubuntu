# Ubuntu

# Usage
Follow these commands one by one in the given order:

* ``ubuntu -u`` - Update & install pkg

* ``ubuntu -i`` - Install ubuntu in termux

* ``ubuntu -l`` - Login in Ubuntu

* ``ubuntu -h`` - Help
